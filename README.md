# zhuhai-weather-underground
珠海地下气象(zhuhai-weather-underground)是一个关注珠海本地天气的个性化气象网站，含实况监测、历史资料、追风板块等内容，可以实时展示气象部门发布的实时气象信息，以及珠海气象爱好者部署在全市范围内个人气象站的数据。
